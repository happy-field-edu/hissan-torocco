# 🚂 ひきざんトロッコ 〜算数大冒険〜

> 小学2年生向け・ブラウザで動く算数アドベンチャーゲーム

公開URL：**https://happy-field-edu.github.io/hissan-torocco/**
リポジトリ：`happy-field-edu/hissan-torocco`（GitHub Pages を利用して無料公開）

---

## 1. アプリの紹介

トロッコに乗ったアヒルのキャラクターと一緒に、**ひっ算（たし算・ひき算）や九九**を解きながら渓谷を進む、一人称視点の算数ゲームです。

- **対象**：小学2年生（34人クラスでの利用を想定）
- **端末**：iPad（タップ操作だけで遊べる）
- **目的**：算数の反復練習を「冒険」「コレクション」「タイムアタック」で楽しくする
- **形式**：インストール不要。URLを開くだけ。セーブは各端末のブラウザ（localStorage）に保存

### 主な遊びの要素

| 要素 | 内容 |
|---|---|
| 🚂 冒険モード | 全10問。3問ごとに背景が険しくなり、10問目は**ラスボス P.O.C. OVERLORD**が登場（最大「30−いくつか」の繰り下がり問題） |
| ⏱️ タイムアタック | 1分間で何問解けるかに挑戦。記録証（リザルト）をスクリーンショットして先生に提出できる |
| ✏️ 入力モード | 「しっかり入力（テンキー打ち込み・報酬100%）」と「えらんで入力（2択・報酬1/3）」を選べる |
| ✖️ かけ算（九九）モード | 先生の「ひみつの呪文」で段を1つずつ解放。授業の進度に合わせて出題範囲が広がる |
| 💎 ジュエル経済＆ショップ | 正解で宝石を集めてジュエルに換金。ハンドル・アクセサリー・お助けアイテムを購入 |
| 📖 おたから図鑑 | 集めたアイテムを一覧・詳細表示するコレクションUI（レアリティ枠・入手日・シルエット表示） |
| 🛡️ お助けアイテム | ふたばシールド（1ミス無効）／ジュエル2ばいチケット |
| 🌈 ふたばフィーバー | 隠しフィギュア装備中に発動。虹色ダイヤモンド（通常の10倍）がドロップ |
| 📝 ふりかえり | 間違えた問題と正しい答えをリザルトで確認。全問正解なら「パーフェクト！」 |

### 先生向けの仕掛け（クラス運営）

- **ひみつの合言葉**：タイトルロゴを2秒以内に5回タップ → 隠しコマンドが出現。先生が黒板でクラス目標を達成したときに合言葉を発表する“アナログ運用”を想定。
  - 背景テーマ解放（例：「ふたば」→おうごんの山、「うちゅう」→宇宙）
  - 九九の段の解放（「くくのいち」〜「くくのきゅう」）
- **記録証**：タイムアタックの結果をスクショしてロイロノート等で提出。
- **隠しフィギュア**：ふたばシールドでパーフェクトを3回連続達成すると「伝説のナビゲーターフィギュア（福田先生のフィギュア）」を解放。

## 2. 技術構成

- **単一HTMLファイル**（`index.html`・約4,300行）にHTML/CSS/JavaScriptを内包。外部ライブラリ・フレームワークなし。
- **CSSのみで擬似3D・各種アニメーション**（コックピット振動／スピード線／岩の急拡大／土煙／紙吹雪／スポットライト 等）。
- **Web Audio API**で効果音を合成（ガチャン・爆発・ファンファーレ・シールド音）。
- **localStorage**でセーブ（ジュエル・所持品・図鑑・解放テーマ・九九の段・入力モード）。
- **画像アセット35枚**：背景・キャラ・景品など。AI生成（Gemini）＋Canvaで透過処理、Python(PIL)で背景除去・トリミング。
- **GitHub Pages**で公開（プッシュすると自動デプロイ）。

## 4. 設計のこだわり（教育的ねらい）

- **学年の範囲を超えない**：問題は常に2年生の学習範囲（繰り上がり・繰り下がりの2桁計算、九九）。九九は合言葉で授業進度に合わせて解放。
- **全員が参加できる**：つまずきやすい子は「えらんで入力（2択）」、しっかり取り組みたい子は「しっかり入力（テンキー）」。報酬に差をつけて挑戦を後押し。
- **間違いを学びに**：誤答した筆算と正しい答えをふりかえりで提示。
- **続けたくなる仕掛け**：ジュエル収集・図鑑コンプ・隠し要素・タイムアタックの自己ベスト更新。
- **先生が主役の運用**：合言葉やクラス目標で、デジタルとアナログ（黒板）を橋渡し。
